# Filtering rules (for AI content creators)

## Must include (Priority 1-3)

### Priority 1: Immediately usable
- Tools / plugins / apps that solve practical problems
- Step-by-step tutorials or guides
- Prompt templates / frames
- Workflow optimization techniques

### Priority 2: Reusable methodologies
- Content creation workflows
- Best practices for using AI
- Productivity techniques
- Skill-building frameworks

### Priority 3: Mindset shifts
- How to think about AI in a different way
- Common mistakes and how to avoid them
- Expert insights into AI usage patterns

## Must exclude
- Hardware / infra (GPU/TPU/etc.)
- Cybersecurity professional content
- Academic papers without practical application
- Funding / revenue / B2B announcements (unless directly useful to individuals)
- Pure benchmarking / model comparisons

## Quality bar
- Test: "A content creator can apply this immediately after reading".
- If no → exclude.
